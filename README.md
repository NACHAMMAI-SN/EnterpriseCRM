# EnterpriseCRM

EnterpriseCRM is a customizable Customer Relationship Management (CRM) platform designed for managing leads, accounts, contacts, and business workflows efficiently.

This project is based on the open-source CRM platform EspoCRM and has been extended and customized for learning, experimentation, and feature enhancement purposes.

---

![Screenshot](https://user-images.githubusercontent.com/1006792/226094559-995dfd2a-a18f-4619-a21b-79a4e671990a.png)

##  Features

* Lead & Contact Management
* Account & Opportunity Tracking
* Role-Based Access Control
* Workflow Automation
* REST API Support
* Customizable Modules
* Clean UI Dashboard

##  Installation

### 1. Clone the repository

```bash
git clone https://github.com/NACHAMMAI-SN/EnterpriseCRM.git
```

### 2. Navigate to project directory

```bash
cd EnterpriseCRM
```

### 3. Configure database

* Create a MySQL database
* Update configuration file with DB credentials

### 4. Run locally

Set up Apache/Nginx and point to project directory.
